# Taxi demand predictor service

## Setup

### Prerequisites

- Python >= 3.11
- [Poetry](https://python-poetry.org/docs/#installation) >= 2.0.0

### Steps

1. **Clone the repository**

   ```bash
   git clone <repo-url>
   cd taxi_demand_predictor
   ```

2. **Install dependencies**

   ```bash
   poetry install
   ```

3. **Activate the virtual environment**

   ```bash
   poetry shell
   ```

4. **Launch Jupyter**

   ```bash
   jupyter notebook
   ```
